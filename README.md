# PyTechTest

## Description
Each `rush.py` implements a function that prints different square patterns using specific characters in Python.

## 📋 Project Overview

| Assignment | Folder       | Pattern                  | Description |
|------------|--------------|--------------------------|-----------|
| 1          | `rush-1-1`   | `o - │`                  | Square with corners `o`, horizontal `-`, vertical `│` |
| 2          | `rush-1-2`   | `/ \ *`                  | Square with diagonal corners and `*` filler |
| 3          | `rush-1-3`   | `A B C`                  | Top corners `A`, bottom corners `C` |
| 4          | `rush-1-4`   | `A B C` (symmetric)      | Top-left `A`, Top-right `C`, Bottom same |
| 5          | `rush-1-5`   | `A B C` (swapped)        | Top-left `A`, Bottom-right `A` (diagonal swap) |

Each `rush(x, y)` function prints the requested square pattern and properly handles:
- Invalid sizes (`x ≤ 0` or `y ≤ 0`)
- Edge cases (`x=1`, `y=1`)
